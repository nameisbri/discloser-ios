# Discloser

A mobile app for managing and sharing STI test results securely.

## Features

- Upload test results with AI-powered extraction
- Track your testing history in one place
- Share results securely via time-limited links and QR codes
- Personalized testing reminders with push notifications
- Risk assessment and testing recommendations
- Dark mode support

## Getting Started

```bash
npm install
npx expo start
```

Requires a `.env` file with Supabase and auth provider credentials. See `.env.example` for required variables.
